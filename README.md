# Latitude

Latitude is a simple gem for calculating distances between two
geographic locations, using great circle math.

## Installation

Add this line to your application's Gemfile:

    gem 'latitude'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install latitude
